# Muscle Load Summary — кто получает нагрузку

## Зачем это нужно
- Быстро понять, какие мышцы и группы получают больше всего работы.
- Увидеть перекос: например, если постоянно прокачиваются только грудь и руки, а спина отдыхает.
- Отличить «стимул» (количество повторов) от «тоннажа» (вес умножаем на повторы), чтобы учитывать разные подходы к тренировкам.

## Источники данных
- Каждая тренировка → список упражнений → сеты.
- У упражнения есть набор мышц с процентами (bundles). Эти проценты показывают, какая часть нагрузки уходит в каждую мышцу.
- Для сетов считаем:
  - **Stimulus** — просто сумма повторов. Работает даже без точных весов.
  - **Volume** — вес умножаем на повторы. Подходит, когда веса указаны.

## Как рассчитываем
1. Для каждого упражнения считаем stimulus и volume.
2. Распределяем эту нагрузку по мышцам пропорционально их процентам в упражнении.
3. Складываем все значения и переводим их в проценты от максимума, чтобы было легко читать.
4. Делаем два отчета: по отдельным мышцам и по группам мышц. Каждый отчет есть и для stimulus, и для volume.
5. Дополнительно считаем `dominance`: какой процент забирает самая нагруженная мышца и две самых нагруженных вместе.

## Как читать результат
- `perMuscle` — список мышц с процентами stimulus.
- `perGroup` — те же данные, но объединенные в группы (например, «Грудь», «Спина»).
- `volumePerMuscle` и `volumePerGroup` — то же самое, но по тоннажу.
- `dominance.top1SharePercent` показывает, насколько одна мышца перегружена относительно остальных.

## Примеры сценариев
- **Новичок без точных весов.** Он заполняет только повторы, поэтому stimulus дает реальную картину, а volume остается пустым — и это нормально.
- **Любитель силовых.** У него всегда заполнен вес. Volume‑карты показывают, что спина получает 45% тоннажа, ноги — только 20%, значит пора добавить приседы.
- **Спортсмен после травмы плеча.** Он видит, что `dominance.top1` резко вырос для грудных, значит нужно вернуться к сбалансированным упражнениям на спину.
